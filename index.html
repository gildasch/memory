Memory Study

Card Catalog

A focused prototype for the card itself: clean white frame, tabletop depth, and a set of flip animations with tiny directional variations so each turn feels a little more human.

Visual direction

Classic tabletop memory, but a little more cinematic.

The back keeps the familiar pale-blue print and crisp border from the reference image. The front uses a simple white mat and saturated nature artwork, while the animation adds a small lift, hinge bias, and soft shadow bloom.

Flip options

Same card, different motion personalities.

Click any card or use the action buttons. Every flip randomizes left or right entry and a slight tilt so repeated turns don’t feel identical.
